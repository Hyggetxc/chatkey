# ChatKey

> A macOS menu bar tool for unifying chat input habits across apps.

## Overview

ChatKey is built to solve one narrow problem well: keep send and newline shortcuts consistent across different chat apps.

It is intentionally small, focused, and early-stage.

## What It Does

- Per-app key mapping
- Send / newline habit normalization
- Lightweight menu bar experience
- Local rule storage
- Accessibility permission guidance
- Simplified Chinese / English UI support

## Why It Exists

Different chat tools often disagree on what `Return` should do.

ChatKey helps users keep one consistent habit without turning the whole keyboard into a remapping project.

## MVP Scope

- Menu bar app
- Per-app rules
- Preset chat mappings
- Custom trigger / output combinations
- Frontmost app detection
- Local persistence
- Basic diagnostics
